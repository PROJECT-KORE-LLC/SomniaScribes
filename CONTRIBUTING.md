# Contributing to SomniaScribes  
A quiet project deserves a quiet approach.

Thank you for your interest in contributing to SomniaScribes.  
This project is small, warm, and intentionally calm — a local‑only journaling sanctuary built and maintained by Project Kore LLC.

Contributions are welcome, but the pace here is gentle.  
No rush. No pressure.

---

## Philosophy  
SomniaScribes is built around four ideas:

- **[Journal](ca://s?q=SomniaScribes_journal_topic)** — a simple place to write  
- **[Local](ca://s?q=SomniaScribes_local_storage_topic)** — your words stay on your device  
- **[Somatic](ca://s?q=SomniaScribes_somatic_topic)** — slow, grounding, atmospheric  
- **[Ambient](ca://s?q=SomniaScribes_ambient_topic)** — warm visuals, soft audio, quiet UI

Any contribution should respect these principles:  
calm, minimal, private, and warm.

---

## How to Contribute

### 1. Fork the Repository  
Create your own fork and work from there.

### 2. Keep Changes Focused  
Small, clear improvements are easiest to review.  
Examples include:

- UI polish  
- Accessibility improvements  
- Theme refinements  
- Performance tweaks  
- Bug fixes  
- PWA improvements  
- Code clarity / cleanup  

### 3. Avoid Feature Creep  
SomniaScribes is intentionally simple.  
Please avoid adding:

- accounts  
- cloud sync  
- analytics  
- notifications  
- anything that breaks the “quiet, local sanctuary” feeling

If you’re unsure, open a discussion first.

### 4. Follow the Style  
The codebase uses:

- clean, readable HTML  
- soft, atmospheric CSS  
- minimal, intentional JavaScript  
- no frameworks  
- no build steps  

Try to match the existing tone and structure.

---

## Pull Requests  
When submitting a PR:

- describe the change in a calm, clear sentence  
- explain why it fits the project’s philosophy  
- keep commits tidy  
- ensure the app still works offline  
- ensure the PWA still installs cleanly  

PRs that maintain the atmosphere and simplicity will be prioritized.

## Asset Notes  
- Background images generated by the developer using Google Gemini  
- Ambient audio sourced from Pixabay (free for use, attribution not required)  
- Do not add copyrighted assets  
- Do not add external tracking, fonts, or CDNs
